# Короткий контекст по заданиям

Этот репозиторий покрывает оба задания Mecenate как один проект.

- Текущая стабильная ветка/состояние: **Task 1 (Feed screen)**.
- Следующий этап: **Task 2** как развитие того же приложения (detail screen, лайки, комментарии, realtime).

Практический смысл:

- `main` остается в рабочем состоянии Task 1 для предсказуемой проверки.
- Task 2 добавляется поверх текущей архитектуры, без разрыва контекста проекта.
